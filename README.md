SampleCsWpfPanel
================

Sample C# RhinoCommon plug-in that demonstrates how to create a tabbed, docking panel that uses WPF.

Building Sample
--------------------
To build the sample, you are going to need:

* Rhinoceros 5 (http://www.rhino3d.com)
* Microsoft Visual C# 2010
